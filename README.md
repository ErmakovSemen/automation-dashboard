# Automation Dashboard

Local landing page with buttons to run scripts and quick links to projects.

## Open the dashboard

- Open `index.html` in your browser.
- GitHub Pages: https://ermakovsemen.github.io/automation-dashboard/

## Customize

- Edit `data.json` to add/remove automations and project links.
- Update `run/*.command` files if script paths change.

## macOS note

The first time you click a `.command` file, macOS may ask for permission.
Allow it to run.
